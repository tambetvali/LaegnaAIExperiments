https://www.getaitool.in/c/ai-app-builder - list of web app generators, quite decent (contained items I had not seen).

https://playcode.io/ - this is simplest *playground* to play with an AI in environment, which reminds the experience of AI app generator as a basic toy or test tool; altough, perhaps you can do the biggest things with tools which look the smallest?

If you want to start a project with hand-written code,
- https://www.pythonanywhere.com/
- Free account is limited to: single web app, 512 MB of storage, and an allowlist of pre-approved external websites.
- In case of my Laegna development, I am using restricted, free services which are not conceptually restricted, but rather providing a small number of MB or processor and robust, working service. This, because I am not creating grand system with big collective: notice that to *handwrite* or even *verified AI-write* 512 MB of code is quite hard, and even small set of logos and images is not much.

These 3 links should actually get one going with some feats:
- App Generators create small applets with your links and task. Many of them support links. Small set of basic applets is often free.
- Playcode.io - indeed one wants to play with HTML, CSS and JS, and to create snippets and test them.
- Pythonanywhere.com - small Flask service can generate flashcards and do many things, 512 MB in terms of how many characters - unless you are creating heavy databases, the modern amounts of memory look almost *huge*, altough you get another level of quality if you have some 100GB - you could run several AI models.
